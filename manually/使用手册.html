Tunnel Proxy — 使用手册

一份面向普通用户的 Tunnel Proxy（macOS 菜单栏应用）友好指南。

Tunnel Proxy 是什么？

Tunnel Proxy 是一款常驻在 Mac 菜单栏（屏幕右上角那排图标）里的小应用。它通过你自己的 SSH 服务器，用一条安全加密的连接来转发你的网络流量。

你可以用它来：

  • 通过位于其他地区的服务器访问互联网；
  • 在不可信的网络（如公共 Wi-Fi）上保护你的流量隐私；
  • 让网站和应用看到你服务器的 IP 地址，而不是你 Mac 的 IP。

它所需的一切都已内置——无需再安装任何其他软件。你只需要知道如何连接一台 SSH 服务器（它的地址，以及你的登录方式）。

工作原理

你的应用把流量发送到 Mac 上运行的本机代理，Tunnel Proxy 通过加密隧道把这些流量送到你的 SSH 服务器，再由服务器转发到互联网。你访问的网站看到的是服务器的 IP 地址，而你的 Mac 与服务器之间的一切都经过加密。

安装应用

  1. 打开你收到的 TunnelProxy.dmg 文件。
  2. 把 Tunnel Proxy 图标拖到 应用程序（Applications） 文件夹上。
  3. 打开 应用程序，双击 Tunnel Proxy。
    • 首次打开时，macOS 可能会询问「确定要打开吗？」——点击 打开。（如果被系统拦截，请前往 系统设置 → 隐私与安全性，向下滚动，点击 仍要打开。）
  4. 菜单栏中会出现一个盾牌图标。应用没有 Dock 图标、也没有窗口——它完全存在于菜单栏中。

提示： 没看到盾牌？可能是菜单栏太满了。按住 ⌘ 拖动一些菜单栏图标腾出空间，或确认应用是否已在运行。

菜单栏弹出面板

点击盾牌图标即可打开主面板。

序号名称作用
①状态彩色圆点和文字表示当前连接状态（见下文）。
②服务器选择要通过哪台服务器连接。
③出口 IP互联网当前看到的你的 IP 地址。仅在已连接时显示。
④连接 / 断开大按钮。开启或停止隧道。
⑤开关常用的开/关选项（见下文说明）。
⑥菜单项打开日志窗口、设置，或退出应用。

读懂状态图标

菜单栏图标的形状和颜色会随连接状态而变化。

图标状态含义
空心盾牌（灰点）未连接流量直接连接——隧道已关闭。
带 + 的左半盾牌（黄点）正在连接 / 正在重连正在建立隧道。
左半填充盾牌（绿点）已连接隧道与代理已启用，一切就绪。
带感叹号的盾牌（红点）出错出现问题。打开 查看日志… 查看原因。

首次设置：添加你的服务器

在连接之前，你需要先把 SSH 服务器信息告诉应用。

  1. 点击盾牌图标 → 设置…（或按 ⌘,）。
  2. 进入 服务器 标签页，点击 ＋ 添加服务器。

填写以下信息：

字段填写内容
名称任意你喜欢的标签，如「家里的服务器」。可选。
主机服务器地址，如 example.com 或 203.0.113.10。
用户名你在服务器上的登录名，如 root。
端口SSH 端口。通常为 22——如无特别说明请保持不变。

选择一种 认证方式——即你如何证明身份：

方式何时使用填写内容
密码你用密码登录你的 SSH 密码
私钥文件你有一个密钥文件（如在 ~/.ssh 中）点击 选择… 选取密钥；若密钥有口令则一并填写
默认密钥 / 代理你的 Mac 已配置好用密钥连接无需填写——自动使用你现有的密钥

点击 保存。

你的密码很安全。 密码和密钥口令保存在 macOS 钥匙串 中——这是 macOS 用来保管你其他密码的同一个安全保险库，绝不会写入明文文件。

你可以添加多台服务器并在弹出面板中切换。在服务器列表里，点击左侧的圆圈即可把某台服务器设为当前使用的服务器。

连接

  1. 点击盾牌图标。
  2. 确认 服务器 一行中已选中正确的服务器。
  3. 点击 连接。

几秒钟后，状态会变为绿色 / 已连接，并显示 出口 IP。这个 IP 就是你服务器的地址——证明你的流量正通过隧道传输。

要停止，点击盾牌图标并按 断开。

日常开关

这些开关位于弹出面板中「连接」按钮的下方。

自动重连（守护进程）

建议保持开启。如果隧道意外中断——比如 Wi-Fi 抖动一下——应用会在后台悄悄为你重新连接。重连时你可能会短暂看到黄色的「正在重新连接…」状态。

显示网络速度

开启后，菜单栏图标旁边会显示实时的上传/下载速度，方便你一眼看到流量在流动。

macOS SOCKS 代理

  • 关闭（默认）： 只有你手动指向代理的应用才会走隧道。
  • 开启： 应用会修改你 Mac 的全局网络设置，让所有应用自动走隧道。

如果你希望 Mac 上的一切都走隧道、而不必逐个配置应用，就把它开启。断开或退出时，如果你想恢复直连，记得把它重新关闭。

登录时启动

开启后，每次登录 Mac 时 Tunnel Proxy 都会自动启动，盾牌图标始终守候在菜单栏中。

设置 → 高级 里还有一个 启动时自动连接 选项。两者都开启，隧道就会在你每次开机时自动建立。

设置项说明

点击盾牌图标 → 设置…（⌘,）打开。共有四个标签页。

服务器

添加、编辑、删除，以及选择当前使用的服务器。（见「首次设置」。）

隧道

  • SOCKS 端口 / HTTP 代理端口——应用在本机监听的端口号（默认 1080 和 8118）。仅在与其他程序冲突时才需修改。
  • 网络服务——macOS SOCKS 代理 开关作用于哪个网络连接（Wi-Fi、以太网……）。通常保持默认即可。
  • 守护进程间隔（秒）——自动重连检查的运行频率。
  • 运行环境——privoxy、ssh、curl 旁的绿色 ✓ 表示应用所需的组件都已就绪。

在这里修改任何内容后，请点击 保存。

高级

  • 登录时启动 与 启动时自动连接（见上文）。
  • 文件——显示配置文件和日志文件的位置，并提供 在访达中显示 按钮。

应用名称和一段简短说明。

查看日志

如果连接失败或行为异常，日志会准确显示发生了什么。

  1. 点击盾牌图标 → 查看日志…
  2. 日志窗口会实时输出：
    • 筛选…——输入关键字只显示匹配的行。
    • 自动滚动——始终显示最新的日志行。
    • 清除——清空显示内容。

日志按颜色区分：绿色表示成功，橙色表示警告，红色表示错误。

常见问题

我还需要安装别的东西吗（Homebrew、其他工具）？
不需要。所有组件都已内置在应用中。ssh 和 curl 是 macOS 自带的。

这是 VPN 吗？
对你的流量来说效果类似，但它是连到你自己服务器的 SSH 隧道，而不是商业 VPN 服务。服务器由你自己提供。

我的密码存在哪里？
存在 macOS 钥匙串——系统的安全保险库里，绝不会以明文形式保存。

它会拖慢我的网速吗？
你的流量会经由服务器多一次中转，因此会有少量额外开销。速度主要取决于你服务器的网络连接。

如何彻底卸载？
先断开连接，关闭 登录时启动 与 macOS SOCKS 代理，然后退出应用，把 Tunnel Proxy 从「应用程序」拖入废纸篓即可。

想知道后台到底发生了什么？查看日志… 窗口永远是排查问题的第一站。
